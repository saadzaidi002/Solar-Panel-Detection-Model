About This Project

Solar Panel Detection & Change Detection AI Model

RESOLVE South  •  SUPARCO Complex, Karachi

This project is an advanced AI system developed to detect and monitor solar panels from high-resolution satellite imagery. At its core, the system leverages a YOLOv8-based machine learning model optimized specifically for Oriented Bounding Box (OBB) detection — enabling precise localization of solar panels regardless of their angle or orientation in aerial views.

The project goes beyond simple detection. A rigorous model comparison pipeline was implemented to evaluate and select the highest-accuracy architecture, ensuring reliable and reproducible results across diverse satellite imagery conditions.

A key feature of the system is its temporal change detection capability. By analyzing satellite imagery from different time periods, the model identifies newly installed solar panels, tracks removals, and quantifies changes in solar infrastructure — providing actionable insights for renewable energy monitoring, urban planning, and sustainability reporting.

  • YOLOv8-OBB model trained on curated satellite solar panel dataset
  • Oriented Bounding Box detection for precise panel localization at any angle
  • Model comparison and optimization for highest detection accuracy
  • Temporal change detection using IoU-based polygon matching
  • Color-coded visualization: added (green), removed (red), persistent (blue) panels
  • Actionable insights for renewable energy monitoring and growth analysis
  • Full-stack web interface for easy model interaction and result visualization
